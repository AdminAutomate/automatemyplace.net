About AutomateMyPlace

What This Site Is

AutomateMyPlace.com is a resource for homeowners who want to build smart homes that work for them — not bill them. We cover home automation hardware, software, and techniques that run locally, don't require cloud subscriptions, and don't become paperweights when a company changes its pricing or shuts down a server.

We focus specifically on the no-subscription angle because it's underserved. Most "smart home" coverage assumes you're fine paying $100–$300/year in ongoing fees. We think that's a bad deal, and we think there are better options for nearly every use case.

What We Cover

  • Home Assistant — setup guides, hardware comparisons, integration tutorials
  • Zigbee and Z-Wave — protocols, hubs, device reviews
  • Smart switches and dimmers — no-neutral wire, three-way, dimming
  • Local security cameras — PoE, wireless, no-subscription storage
  • Whole-home automation — locks, thermostats, sensors, DIY security
